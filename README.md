## 项目简介

本项目使用 Python 和 Taichi 框架实现了贝塞尔曲线的实时绘制。通过鼠标交互添加控制点，程序能够实时计算并渲染出平滑的贝塞尔曲线，展示了计算机图形学中曲线生成和光栅化的基本原理。


## 实验目标

- 理解贝塞尔曲线的几何意义
- 实现 De Casteljau 算法计算贝塞尔曲线上的点
- 掌握光栅化的基础概念（像素操作与坐标映射）
- 实现现代化的图形界面交互（鼠标点击、键盘事件）

## 数学原理

### De Casteljau 算法

De Casteljau 算法是一种通过递归线性插值计算贝塞尔曲线上点的方法：

1. 线性插值：对于参数 t ∈ [0,1]，在相邻控制点之间进行线性插值
   P'(t) = (1-t)P_i + tP_{i+1}

2. 递归计算：重复线性插值过程，直到只剩下一个点

3. 曲线生成：该点即为贝塞尔曲线在参数 t 处的坐标

### 算法示例

对于 3 个控制点 P0, P1, P2：

- 第一层：P01 = (1-t)P0 + tP1, P12 = (1-t)P1 + tP2
- 第二层：P = (1-t)P01 + tP12
- 最终点 P 即为曲线上的点
